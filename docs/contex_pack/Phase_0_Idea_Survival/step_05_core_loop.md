✅ CORE LOOP EXPANDED (WITH PSYCHOLOGY)

Step 1: Trigger (Natural, Weekly)

Trigger event:
	•	User introduces 2 people
	•	Then… silence
	•	User thinks:
“Did they connect? Did I waste that intro? Do I follow up again?”

✅ This happens weekly for recruiters, founders, BD people.

⸻

Step 2: Action (10 seconds)

User creates intro as usual, but adds one line:

“To make follow-through easy, here’s the Truferral intro receipt link (optional): [link]”

✅ Works in:
	•	Email intros
	•	LinkedIn DMs
	•	WhatsApp messages
	•	Slack
	•	Text message

No new workflow. Just one habit.

⸻

Step 3: Instant Value (Receipt Created)

The link opens a simple receipt:

Truferral Intro Receipt
	•	Intro created by: Rehan
	•	Person A: Mike
	•	Person B: David
	•	Date/time
	•	Channel
	•	Status: Not started / Replied / Call scheduled / Completed / Not a fit

Key design rule:
✅ neutral language (no pressure, no awkwardness)

⸻

Step 4: Outcome Updates (Lightweight)

Either person updates with 1 tap:
	•	✅ “Replied”
	•	✅ “Meeting booked”
	•	✅ “Not a fit”
	•	✅ “No response yet”

No essays. No CRM.

⸻

Step 5: Reward (Closure + Reputation Safety)

Introducer gets closure:

✅ “Meeting booked” → feels win
✅ “Not a fit” → still feels closure
✅ “No response” → system handles reminder (later)

Reward = closure + reduced anxiety + less awkward chasing.

This is what forms the habit.

⸻

Step 6: Retention Loop (Repeats Automatically)

Every time user makes another intro, they repeat:
	•	“Drop receipt link”
	•	“Stop manually chasing”
	•	“Get closure”

Habit forms because the trigger is external (intros happen anyway).
